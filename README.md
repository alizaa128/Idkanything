# Weather Dashboard App

## Description
The Weather Dashboard App allows users to view current weather conditions and forecasts for various locations. Users can search for cities and receive real-time weather updates.

## Features
- Search for weather by city name
- Displays current temperature, humidity, and wind speed
- 7-day weather forecast
- Responsive design for mobile and desktop 

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API for fetching weather data

## Installation
1. Clone the repository: `git clone https://github.com/alizaa128/Idkanything.git`
2. Navigate to the project directory.
3. Open `index.html` in your preferred web browser.

## Usage
- Enter a city name in the search bar to get the current weather information.
- Click on the forecast tab to view the 7-day weather forecast.
